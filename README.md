# TabSense - Context Preservation Platform

TabSense is a developer-focused tool designed to preserve browsing context during research sessions, preventing tab clutter and context loss.

## Monorepo Project Structure

* `/tabsense-backend` - Spring Boot REST API (Port 8080)
* `/tabsense-dashboard` - React Single Page Application (Port 5173)
* `/tabsense-extension` - Chrome Extension (Manifest V3)

## Technical Stack

* **Backend**: Java 17+, Spring Boot 3.x, Spring Data JPA, Spring Security (JWT), PostgreSQL
* **Frontend**: React (Vite, TypeScript, TailwindCSS/Vanilla CSS)
* **Extension**: Manifest V3 (Vanilla JS, Service Workers)
